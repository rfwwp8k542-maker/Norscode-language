Om Norscode

Norscode er laget for å være et tydelig norsk språk for små og mellomstore verktøy, demoer, studier og produkter. Vi vektlegger lesbarhet, statisk struktur og en praktisk arbeidsflyt fra kildekode til kjørbar binær.

Språk

Norscode bruker norsk syntaks, funksjoner, betingelser, løkker, lister og tydelige standardfunksjoner.

Verktøy

CLI, Studio, test-runner og standardbiblioteket gjør det mulig å jobbe i én sammenhengende arbeidsflyt.

Bygg

Språket kompilerer til C og kan også pakkes som standalone binary for enklere distribusjon.

Mål

Målet er å bygge et språk og et økosystem som er lett å lese, lett å vedlikeholde og lett å bruke.

Kort kommandoreferanse

norcode run app.no
norcode check app.no
norcode test
make binary

Studio i praksis

Norscode Studio er laget for å gjøre det enklere å navigere i koden, finne symboler, se tester og holde styr på åpne filer og nylig lukkede faner.
