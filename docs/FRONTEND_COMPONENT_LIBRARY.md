# Frontend Component Library

Et komponentbibliotek samler de viktigste visuelle byggesteinene.

## Mål

- gjøre UI gjenbrukbart
- holde knapper, inputs og kort konsistente
- redusere duplisering mellom sider

## Komponenter

- knapp
- input
- select
- kort
- panel
- dialog
- badge
- navigasjon
- tabell

## Regler

- komponenter skal bruke design tokens
- komponentene skal ha tydelige props
- komponenter skal være enkle å teste
- komponentene skal fungere på små og store skjermer

## Når dette er ferdig

- nye sider kan bygges raskere
- visuelle forskjeller blir mindre tilfeldige
