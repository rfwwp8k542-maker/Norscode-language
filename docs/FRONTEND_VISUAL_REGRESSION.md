# Frontend Visual Regression

Visuell regresjon hjelper oss å oppdage layout- og designendringer.

## Mål

- fange uønskede visuelle endringer
- beskytte komponentbiblioteket
- gjøre frontend-justeringer tryggere

## Modell

- ta snapshots av viktige sider og komponenter
- sammenlign mot godkjente referanser
- fokuser på kritiske views

## Regler

- ikke snapshot alt
- hold referansebilder ryddige og få
- oppdater referanser bevisst, ikke automatisk

## Når dette er ferdig

- visuelle brudd oppdages tidlig
- UI-endringer er mindre risikable
