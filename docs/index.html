Et norsk programmeringsspråk som kompilerer til C.

Norscode er laget for å være lett å lese, lett å teste og lett å bygge videre på. Du kan kjøre det som CLI, bygge en standalone binary, og hente ferdige nedlastinger fra siste release.

Last ned siste release Se nedlastinger Se kildekode

Hva det er

Et språk og en compiler-kjede for norsk syntaks, statisk typing og selfhost-komponenter.

Hvordan det distribueres

Standalone binary for alle tre hovedplattformene, publisert som release-vedlegg.

Språket

Norscode støtter norsk syntaks, statisk typing, lister, pakker og en stadig mer komplett compiler-kjede.

Selfhost

Parser, AST-bro og selfhost-løp er holdt i sync, slik at prosjektet kan vokse uten at to implementasjoner sklir fra hverandre.

Nedlastninger

Ferdigbygde binærer publiseres for Linux, macOS og Windows, og kildekoden ligger åpent i GitHub-repoet.

Bygg selv

Hvis du vil lage din egen binary lokalt, kan du bygge den med PyInstaller via repoets eget script.

python3 -m pip install -r requirements-dev.txt make binary ./dist/norscode test

På Windows blir filen dist/norscode.exe.

Kort status
