Kontakt

Hvis du vil komme i kontakt med prosjektet, foreslå en forbedring eller melde en feil, er den enkleste veien via prosjektets egen kodebase og GitHub-flyt.

GitHub

Åpne en issue eller pull request i repoet for forslag, feil og forbedringer.

Dokumentasjon

Les README, hjemmesiden og personvernsiden for å forstå hvordan prosjektet er bygget opp.

Studio

Norscode Studio brukes til å redigere og vedlikeholde prosjektet lokalt.

Nettsted

Denne statiske siden viser det viktigste om språk, studio og bruk av Norscode.

Hva du kan sende

Rapporter feil, idéer til nye standardbibliotekfunksjoner, forbedringer i Studio, eller forslag til bedre dokumentasjon og eksempler.

Hva vi foretrekker

Korte, konkrete meldinger med eksempler gjør det lettere å følge opp og forbedre språk og verktøy raskt.
