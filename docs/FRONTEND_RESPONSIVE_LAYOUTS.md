# Frontend Responsive Layouts

Frontend må fungere godt på små og store skjermer.

## Mål

- holde layout lesbar på mobil
- utnytte plass på desktop
- unngå separate design for hver skjerm

## Modell

- bruk flytende grid og fleksible paneler
- bryt layout ved smale skjermer
- hold spacing og typografi skalerbar

## Regler

- én kolonne på smale skjermer når det passer
- to eller flere kolonner på større skjermer
- bilder og tabeller skal kunne tilpasses

## Når dette er ferdig

- appen ser bra ut på mobil, nettbrett og desktop
